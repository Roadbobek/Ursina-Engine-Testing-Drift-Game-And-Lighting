# Ursina Engine Testing Drift Game And Lighting
A couple Ursina engine test programs, a car drift game, a lighting test, and some more stuff. Made 29/5/2025.

This was built with the development version of Ursina, pip install `pip install https://github.com/pokepetter/ursina/archive/bc8de88d26b0b7f614ba9c2d138dd1cfee9d1aeb.zip`
